
Docker build contains the Database, and nodejs server.

Linux (ubuntu):
1.) Install docker
    $ ./install_docker.sh
2.) Build docker image:
    $ ./build_docker 
3.) Run server:
    $ ./run_docker-dev.sh
4.) Done?
    $ sudo docker kill nodejs_server
